# DSA-PROJECT-GROUP-19
Social Graph
1. Problem Statement

Design a system that models relationships between users in a social network. The system should allow users to connect, view mutual friends, and suggest new connections using graph-based algorithms.

2. Key Features
   
User creation and profile storage
Add/remove friendships (edges in graph)
View friends list
Mutual friends detection
Friend recommendation (based on graph traversal)
Shortest connection path between users (BFS)

3. How to run
   requirements-Visual Studio Code
   steps-save and run the html codes ie SocialGraph.html
       -Click on GO Live

ELEVATOR ASSIGNMENT SYSTEM
1. Problem Statement

Design a system that efficiently assigns elevators to requests in a multi-floor building, minimizing wait time and optimizing elevator movement.

2. Key Features
Multiple elevators management
Request handling (up/down calls)
Optimal elevator assignment
Direction tracking (up/down/idle)
Floor scheduling
Queue-based request handling
